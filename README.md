Irunan World Map - Copy Source - Creation By Kronus Genovius
Đây là giả lập làm bằng RPG Maker do Kro tự tạo ra. Ở đó, các bạn sẽ được nhập vai Ironus-sama để đi khám phá thế giới Irunan!! OwO
Các File trong Folder Game có thể bỏ vào RPG Maker MV đã được mình Decrypter
[Khi tải về và Extract ra,mở bằng RPG Maker MV thì bạn sẽ làm gì được ?]
1.Có thể chỉnh thông số,Model,v.v (Khuyến cáo:Đừng tự ý chỉnh những file hoặc trong Database >> System vì File thì sẽ có riêng 1 mật mã được gọi là EncrytionKey,mật mã này có thể tìm trong *Nơi chứa thư mục game*/data/System.json,còn Đừng tự ý chỉnh Database >> System thì thử đi sẽ biết =))))
2.Có thể tự do sáng tạo hơn
(Và còn nhiều thứ khác bạn có thể làm trong RPG Maker MV)
